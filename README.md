# Welcome to StreamOS

## Running StreamOS

### On Your Computer

1. Install **Visual Studio Code** if you haven't already.
   - Download it from https://code.visualstudio.com/
2. Open the **Releases** section of this repository.
3. Download the version you want as a **.zip** file.
4. Extract the ZIP file.
5. Open the extracted folder in **Visual Studio Code**.
6. Open a terminal:
   - Select **Terminal → New Terminal**.
7. Run the following command:

   ```bash
   npx expo start
   ```

8. Wait for the Expo development server to start.

## On Your Phone

1. Install **Expo Go** from the App Store or Google Play.
2. Make sure your phone and computer are connected to the **same Wi-Fi network**.
3. Open **Expo Go**.
4. Select the **StreamOS** project from the list, or scan the QR code shown in the terminal if needed.
5. Enjoy StreamOS.

---

# Closing StreamOS

## Option A (Recommended)

1. On your phone, press and hold the screen with **three fingers**.
2. Tap **Home** to exit the app.
3. On your computer, return to the terminal.
4. Press **Ctrl + C** to stop the Expo development server.

## Option B

1. Close the **Expo Go** app.
2. Close **Visual Studio Code**.
